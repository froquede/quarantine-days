# quarantine-days
> one sketch every quarantine day

- rules:
  - every sketch outputs image or video / gif
  - no more than 1h for one sketch
  - 1 folder per day
  - shared dependecies

- getting started:
  - fork this repository
  - npm install
  - npm start
