# quarantine-days
> one sketch every quarantine day

- rules:
  - every sketch outputs image or video / gif
  - no more than 1 hour per sketch
  - organize days in folders
  - shared dependecies

- getting started:
  - fork this repository
  - npm install
  - npm start
